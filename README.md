# CClipboard

一个简洁的 macOS 剪贴板历史管理工具，常驻菜单栏，随时查看和复用历史复制内容。

## 功能

- **自动记录** — 实时监听系统剪贴板，自动记录复制的文本
- **历史列表** — 菜单栏下拉查看，支持搜索过滤
- **一键复制** — 点击条目复制到剪贴板，窗口自动关闭并回到原应用
- **回车粘贴** — 选中后按回车，自动复制并粘贴到当前对话框
- **固定条目** — 常用内容可固定，不受数量上限影响
- **全局快捷键** — `⇧⌘V` 随时唤出历史面板

## 安装

### 从 Xcode 构建

```bash
git clone https://github.com/Geek-Johnny/CClipboard.git
cd CClipboard
open CClipboard.xcodeproj
```

## 使用

| 操作 | 说明 |
|------|------|
| 点击菜单栏 `📋` 图标 | 打开剪贴板历史 |
| `⇧⌘V` | 全局快捷键打开浮动面板 |
| 点击条目 | 复制内容 → 自动关闭 |
| `↑` `↓` + `Enter` | 选择并粘贴到原应用 |
| `Esc` / `⌘W` | 关闭面板 |
| 鼠标悬停 `📍` 按钮 | 固定/取消固定条目 |
| 搜索框 | 实时过滤历史记录 |
| 底部 `🗑 清除全部` | 清空历史 |

## 系统要求

- macOS 14.0+
- Xcode 26.5+（开发）

## 技术栈

- SwiftUI + AppKit
- 纯原生开发，无第三方依赖
- JSON 文件持久化
- Carbon 全局快捷键
